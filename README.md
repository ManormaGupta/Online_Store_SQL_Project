# Online Store SQL Project 🛒

## 📌 Project Overview
A complete end-to-end MySQL project 
analyzing an online store database 
with 50 users dataset.

## 🛠️ Tools Used
- MySQL Workbench
- SQL

## 📂 Project Structure
- database_setup.sql — Database & Tables
- insert_data.sql — 50 Users Data
- views.sql — SQL Views
- stored_procedures.sql — Procedures
- functions.sql — Custom Functions
- triggers.sql — Automated Triggers
- analysis_queries.sql — Business Insights

## 🔍 Key Features
- Tables with Foreign Keys
- Views for easy data access
- Stored Procedures for data entry
- Functions for Salary Grade & Age
- Triggers for automatic logging
- String & Date Functions
- Backup & Restore

## 📊 Key Insights
- City wise user distribution
- Gender wise salary analysis
- Age group analysis
- Salary grade classification
- Top 5 highest salary users
